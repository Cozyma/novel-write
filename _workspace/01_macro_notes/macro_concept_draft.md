# マクロコンセプト草案：表向きは底辺冒険者、裏では最強の暗躍者

## 1. ログライン（一言あらすじ）
政治的都合で抹消された「人体実験部隊」の生き残りが、辺境のギルドで念願の「底辺冒険者（スローライフ）」を満喫しようとするが、規格外の実力が漏れ出てしまい、意図せず裏社会のVIPたちから畏怖・重用されてしまうギャップ・ファンタジー。

## 2. コアとなる「報酬（カタルシス）」
- **ギャップ・勘違い（優越の報酬）**:
  表の冒険者たちは主人公を「冴えない下っ端」と見下しているが、裏社会のドンや最強クラスの戦士たちは主人公の異常な実力を知り、冷や汗を流して敬語を使う。この「読者だけが知っている優越感」をメインにする。
- **無双（力と問題解決の報酬）**:
  表の生活（スライム狩りなど）を邪魔する強大な敵や裏社会の脅威が現れた時、面倒くさそうに「昔の力」を少しだけ解放し、一瞬で蹂躙する爽快感。

## 3. 主人公の設定（仮）
- **出自**: 国家の暗部に存在した非人道的な「人体実験部隊（仮：第零機関など）」の最高傑作。しかし、部隊の力が強大になりすぎたため、恐れた上層部の政治的判断で部隊ごと抹消された。
- **現状**: 奇跡的に生き延び、辺境の街に流れ着く。「生きて、温かい飯が食えて、自分の布団で寝られる」という最低限の平和な生活に異常な幸福を感じており、絶対に目立たずEランク（底辺）冒険者として平和に生きることを目標にしている。
- **能力・戦闘スタイル**:
  - **AI型人工精霊との同期**: 脳内に紐づいた人工精霊による膨大な瞬間思考、敵魔法の秒速解析、およびミリ単位の精密動作のサポート。
  - **暗殺特化（片手剣と投擲）**: 派手な魔法をぶっ放すのではなく、精霊の演算による「最適解」に従い、敵の急所にナイフを投げ、片手剣で急所を貫く「無駄のない一撃必殺」スタイル。
  - **切り札（代償と大出力）**: 身体能力の限界突破（反動あり）。また、外部リソース（高ランクの魔石など）を消費することで、本来の規格外である「戦術級の大出力」を一時的に引き出せる。

## 4. 序盤の展開イメージ（フック）
1. **正規ギルドでの門前払い**: 辺境の正規ギルドに登録しようとするが、身元不明（または魔力測定時のエラー等）で門前払いを受ける。生きるために、治安の悪い「アウトローギルド（裏ギルド）」で下っ端の雑用係として働き始める。
2. **圧倒的戦力との遭遇（偶然の介入）**: アウトローギルドの女ボスが、敵対組織の雇った「凄腕の剣士（界隈で有名な実力者）」に襲撃され、絶体絶命の危機に陥る。主人公は「せっかく見つけた平和な職場（雑用係）を潰されては困る」と偶然（または仕方なく）介入。
3. **実力の露呈とマーク**: 人工精霊の演算と精密な一撃で、凄腕の剣士を瞬殺。助けられた女ボスは、主人公の異常な強さに底知れぬ恐怖と畏敬の念を抱く。
4. **二重生活の始まり**: 主人公は表面的には「アウトローギルドの冴えないパシリ（下っ端）」として扱われ、先輩チンピラからアゴで使われる日々をエンジョイする。しかし裏では、女ボスから直接「誰も解決できない極秘・高難度ミッション」を泣きつかれ、こっそり処理する暗躍者となる。
