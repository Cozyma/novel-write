　冒険者ギルドの本館に入るのは、これで三度目になる。一度目は紙の山を抱えて、二度目は突き返された紙を抱えて。三度目の今日は、紙一枚だ。灰の斡旋所の割印がひとつ押してあるだけの、薄い一枚。〔E1〕

　それで、通った。

　昨日あれだけ書き込んで駄目だった扉が、今日は割印ひとつで開く。書類の出来で言えば百対一で昨日の勝ちだが、通る通らないは出来の話じゃないらしい。入り口の検めは俺の顔も見なかった。見たのは割印だけだ。よく出来てるんだか、何なんだか。

　街道は明日からだが、その前に済ませるものが二つある。依頼の手続きと、雇い主との顔合わせだ。フロアに入って三歩で、癖が先に仕事をした。出口は三つ。正面の大扉、東の資材搬入口、その奥の職員用。数え直してから、受付の列に並ぶ。

　声は、十も数えないうちに掛かった。

「おい、新顔」

　振り向くと、装備のいい若いのが三人。仕事前というより、仕事を探しに来て見つからなかった顔をしている。真ん中の一人が、顎で俺の手元——割印の紙を指した。

「下請けか。ちょうどいい、荷運びの口が空いてる。北倉から東門まで木箱で二十。相場の三分でどうだ」

　後ろの二人が笑った。相場の三分の一で人を使えるなら、そりゃ笑いも出る。

　断り方を間違えると、面倒なのはこっちだ。連中は揉めても失うものがない。こっちは揉めた時点で案件が飛ぶ。下請けの立ち入りってのは、雇い主と案件があって初めて成立している。この場の喧嘩は、値段がまるで合わない。

　だから、値段の合わない客にする顔をした。困った風に眉を下げて、要領を得ない返事で話を間延びさせる。あー、いや、俺は今日、手続きがあって。荷運びは、その、腰が。——銭にならず、話も通じない。そう見切らせるのが一番早い。事実、真ん中のやつが早くも欠伸を噛み殺しはじめた。

　ところで連中の笑い声には、天井があった。笑いはするが、フロアに響かせない程度に絞ってある。誰への遠慮かと視線の先を追うと、フロアの奥、階段の脇に、文官服の男が立っていた。書類を片手に職員と何か話している。それだけの姿なのに、あの一角だけ空気が静まっている。〔E2〕

「規程と公正の体現者、だとさ」

　列の前にいた下請けらしいのが、聞いてもいないのに小声で教えてくれた。ギルドの管理職で、規程に一番うるさく、一番公正な人なんだそうだ。へえ、と返しておいた。その公正の行き届いたフロアで、俺は今しがた相場の三分の一で買い叩かれかけたわけだ。血筋と立場の値付けだけは、公正の内側で当たり前のように回っているらしい。

「——なんだ、楽しそうだな。混ぜろよ」

　声は、三人の真後ろから降った。

　まずいのは、俺がそれまで気づいていなかったことだ。出口を数え、気配を拾って生きてきた。その俺が、五歩の距離に入られるまで、男の接近をまるで拾えなかった。足音も、衣擦れも、視線の圧も、何も無かった。……そういう歩き方をする人間を、俺は夜の仕事でしか見たことがない。

　三人が振り返って、固まった。

　立っていたのは、研がれた剣をそのまま人の形にしたような男だった。思ったより若い。人好きのする顔に、軽い実戦装備。腰の長剣に飾り気はなく、柄だけが使い込まれて色が変わっている。

「暇なんだろう？　なら稽古をつけてやる。実戦形式で」

　男が笑って言い、三人は「いえ」「自分らは急ぎで」と後ずさって、みるみる人混みへ溶けた。周りは静まるどころか、逆に沸いた。近くの卓の冒険者が男に声を飛ばし、男が軽口で返し、笑いが起きる。奥の一角が『静まる圧』なら、この男の周りは『沸く圧』だ。同じ建物の中に、圧が二種類ある。〔E3〕

　最初の一声を聞いた時から、嫌な照合が始まっていた。

　歩き出しの重心が、右に乗る。

　〈歩幅、重心、一致〉

　セリファが短く告げた。分かってる。——昨夜、路地で退路を読み合った、あの暗殺者だ。表の顔は、ギルドの高位冒険者ときた。〔E4〕

　男の目がこっちを向いた。目の奥で、一瞬だけ何かが検算された。上背、手の位置、立ち方。俺が昨夜やったのと同じ照合を、この男は一瞬で済ませて、済ませた計算の速さごと、口元の緩みの下に仕舞った。

　初対面の顔を、互いに作る。

「いい目をしているな、お前」

　男は新顔を値踏みする体で——つまり完璧な芝居で——そう言った。それから、周りに聞こえる声の張りで続けた。

「俺はこの男を買っている。覚えておくといい」

　フロアの視線が一斉にこっちへ来た。勘弁してくれ。

「買われるほどのことは、した覚えがありませんよ」

「そうか？　俺は物覚えがいい方でね」

　それで、取り引きは済んだ。あんたの夜を俺は覚えていない、俺の夜もあんたは覚えていない——互いの手の内は、互いの懐の中。一言ずつで、確かめ合ったのはそれだけだ。ここで下手を打てば、場に出るのは俺の正体だけじゃない。向こうの夜の顔も一緒に出る。だからこの均衡は崩れない。崩さないことが、そのままお互いの商売の続きになる。

　ついでに、こうも読める。この男は昨夜の落とし前に、褒め言葉で保険を掛けにきた。得体の知れない相手と敵対しないための、一番安い掛け金——それが公認だ。そして裏を返せば、俺の正体を知っている男が、この先もギルドの中枢に立ち続けるということでもある。公認ってのは、監視の別名らしい。

「ヴィクテル様が……新人を?」

　職員の囁きが聞こえて、フロアのざわめきが一段上がった。ヴィクテル。男は言うだけ言って片手を上げ、振り返らずに歩き去った。〔E5〕

　効果は、思ったより速く回った。

　受付へ戻ると、俺の番が来ていた。というより、列が消えていた。さっきまで前に並んでいたはずの連中に、どうぞお先にと譲られた。窓口の職員は敬語になっていて、三枚要ると言われていた手続きの書類は、一枚で済んだ。残りの二枚がどこへ行ったのかは、誰も説明しなかった。〔E6〕

　評判ってのは、履歴より速い。ひと晩かけて清書した昨日の俺の身にもなってほしい。

「ノルク様。雇い主の方が、お見えです」

　顔を上げて、すぐに分かった。取次の指す先を確かめるまでもなかった。

　フロアの真ん中に、道が出来ていた。

　人混みがひとりでに割れて、そこを女が歩いてくる。正規冒険者への会釈は、ある。立場への挨拶だけは、皆きっちりやる。ただ、声がどこからも飛ばない。ヴィクテルの周りは沸いた。奥の管理職の周りは静まる。この女の周りは——下がる。温度が、一段ずつ。

　きっちり結い上げた濃色の髪。制式装備の着こなしに一分の隙もなく、なのにその制式が、体格に対してどこか窮屈そうに見える。背筋は定規で引いたようにまっすぐだ。歩き姿だけなら、フロアの誰より絵になる。実際、皆が見ている。器量への遠巻きの視線——最初は、そう読んだ。

　違った。

　全員、目だけを見ていない。視線が、瞳の高さの手前で滑って逸れる。顔は見る。装備も見る。目の前で、折り返す。フロア中が、申し合わせたようにそれをやる。

　だから俺は、真っ直ぐ目を見た。人と話すのに、他にどこを見ろと言うんだ。

　深い、琥珀金だった。

　——《濁り血》。瞳に出るという、あれか。

　札の隅で消されていた二文字が、音を立てて嵌まった。……どこが濁っているんだか。嫌になるくらい澄んだ色だ。濁っていると呼んだ側の目の方が、よほど濁っている。

　綺麗な目だと思った。そう思ったのは、たぶん、この場で俺だけだった。

　立場だけは、ある。だから道は空く。敬意だけが、無い。全員が当たり前の顔で下に見ている——それをフロア中が、上品にやっている。

　目が合った女が、一瞬、瞬きを止めた。〔E7〕

「マーレン・エスリッド。正規冒険者です」

　止まった一拍を、女は姿勢のいい名乗りで上書きした。声まで背筋が通っている。

「ノルク。灰の斡旋所から来た便利屋だ」

「条件を確認します」

　世間話の入る隙は、髪の結い目ほどもなかった。

「行程三日、街道筋の調査と護衛。集合は明朝、六の鐘。装備は各自。危険手当は等級二で計上します。報酬は前払い三割、残りは完了査定の後に」

　断定調の立て板に水だ。ただ、中身を聞いて、おやと思った。危険手当の等級も前払いの割合も、規程の上限側に張り付いている。下請けの条件なんてのは雇い主の裁量でいくらでも削れるのが相場で、現に今朝、俺は相場の三分の一を提示されたばかりだ。

「ずいぶん公正だね。下請け相手に」

「下請けだから削っていいという条項は、存在しません」

　規程に無いものは、存在しない。そういう言い方を、この女は皮肉じゃなく素でやるらしい。

「六の鐘だったね。五の鐘半には出られるよ、俺は」

「六で結構。遅れたら置いていきます」

　融通は要らない、という顔だった。それから女は、事務の続きと同じ声で言った。

「降りるなら手付けを返して、今日のうちに。——今月で三人、道中で消えた」

　受付板の、空いた釘を思い出した。掛かって、残って、外された札。三人。今月だけで三度、この女は夜の道の途中で雇い人に消えられている。声は事務のままだったが、その一文だけ、置き方がわずかに重かった。値踏みの目がこっちを見ている。四人目かどうかを見定める目だ。

「三人とも、夜に?」

「いずれも、夜に」

「なら、夜番は俺が持つよ」

　女の眉が、わずかに動いた。

「夜番ってのは、逃げる気のある奴には一番いい持ち場だろ。逃げるつもりなら、初日の夜に白黒がつく。あんたは三日疑いながら歩かなくて済むし、俺は三日値踏みされずに済む。楽だ、お互いに」

　疑いを晴らそうという話じゃない。試験があるなら、前倒しにしようという話だ。わざわざ試されに行く四人目は、想定になかったらしい。女の値踏みが処理に詰まるのが見えた。目が二度、俺の顔の上で行き場を探した。

「……好きにしなさい」

　それで実務は終わりだった。終わりの、はずだった。

「さっきから、何を見ているの」

　考えるより先に、口が出た。

「いい目だと思ってね」

　顎が、わずかに引かれた。

　殴られ慣れた人間の反射だ、あれは。

　しまった、と思ったときには、もう遅い。

　空気が凍った。

「……ふざけているの?」

「本気だよ。本気で楽をしたいだけだ。——夜番も、目の話も」

　琥珀金から、温度が抜けていった。苛立ちが、その一段下の冷えに変わるところまで、目を見ていたから全部見えた。女は何も言わず、規程通りとしか言いようのない角度の一礼をして、踵を返した。〔E8〕

　去っていく背中を、フロア中の視線が見送った。最後まで、誰も目は見ないままで。

　〈雇用主の規程遵守率——実測を、開始します〉

「実測するな」

　言ってから顔を上げて、気づいた。

　フロアが、静まり返っていた。

　目の話は、聞こえていた。

　視線の種類なら、商売柄いくらでも読んできた。畏れ、値踏み、殺気。今この場に満ちているのは、そのどれでもない。強いて言うなら——崖っぷちで逆立ちを始めた男を見る顔、だ。憐れみを一段通り越した、静かなドン引き。どこかで誰かが、書類の束を取り落とす音がした。

　どう映ってるのかは、想像がつく。つくから、それ以上は考えない。

「……こういう朝の後は、決まって、面倒が増える」

　経験則だ。占いより当たる。

　明朝、六の鐘。三日の街道。温度の抜けた琥珀金と、買われた覚えのない公認と。俺は手続きの済んだ紙を懐に入れて、頭を抱えた。〔E9〕

3. **E3 いびり再開の一拍を欠落（regen側のレンダーミス）**——E3冒頭「いびりが一拍続く（嘲笑の追加）中」は input に存在するが、generation_blocks の block2 fact には無く、regen は欠伸で萎んだ流れのまま介入させた（中断される進行の弱体化）。**還流候補③: block2 fact へ「いびり一拍再開」を同期**（timeline と blocks の粒度ドリフト）。
4. **E9 締めの物理動作**——E9正本『頭を抱えて退場』に対し、確定ドラフトは「頭を抱えたくなるのを堪えて」＝逆生成時の軽微な非可逆（正本と本文の不一致が既に存在）。regen は正本通り「頭を抱えた」。人間判定時にどちらかへ寄せる。

### (b) 台詞・ビートレベル（fixed_line/予約語以外の発明差）
- **一致**: 符牒一往復・「いい目をしている」対句・保険/監視の内心・書類三枚→一枚・崖っぷちの逆立ち・書類の落ちる音——逆生成済み event_timeline の記述密度が高い箇所はほぼ台詞レベルまで収束した。
- **正本のみの発明（inputに無い＝人間フィードバック期の発明層）**: 「規程では交代制です」「じゃあ交代制で、俺の番だけ長めに」の押し返し一往復／「異議は?」の締め方／「嘘だろうこの人」／厠の汲み替え＋「困った顔の在庫」／「一言で、場の値段が変わった」／「立て板に規程」／「こいつの庭が、広いんだ」／ヴィクテルの喉の奥の笑い（speech_rule にあるが regen は未使用）／締めの「楽をしたい人生の予定表」。
- **regenのみの発明**: 北倉→東門・木箱二十／「規程に無いものは、存在しない…素でやるらしい」／「占いより当たる」。
- **regen側の取りこぼし**: E8注記『本人比で本気なのに声が軽い』のテクスチャ（正本＝「本気で言ったんだが、こういう時に限って、声が軽くなる」の地の文）を台詞のみで処理——input にあった要素のレンダー漏れ。

### (c) 文体・温度レベル
- 基調（乾いた一人称・短い断定・内心の損得語り）は一致。文体アンカー＋レンダー規律で声の再現は成立している。
- 正本の方が**中盤の機知の密度が高い**（在庫を数え直す・場の値段・立て板に規程 等、いびり〜条件提示の間に小さな笑いが1段落1個ある）。regen はやや乾き過ぎ・符牒後の均衡説明がわずかに説明過多。
- 感覚のメタファー（「焚き火の芯みたいな色」等）は再現不能な残差＝人間判定・改稿で足される層と確認。

### 結論（正本の穴＝人間フィードバックでしか入らなかった要素）
- **構造は固定済み**: 逆生成 event_timeline＋fixed_line＋POV規律の3点は再現性を担保する（誤爆・転回・符牒は独力で再現）。
- **穴は3種**: ①台詞断片の話者・論理の未記載（還流候補①②）②timeline と generation_blocks の粒度ドリフト（還流候補③）③ビート間の機知・メタファーの密度＝正本化不能の残差（ここが人間判定の恒常的な領分）。
